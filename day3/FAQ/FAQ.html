1.What is HTML and what is it used for?

HTML, or Hypertext Markup Language, is the standard language used to create and design web pages and web applications. It provides the structure for web content by using a system of tags and attributes to define elements like headings, paragraphs, links, images, and other types of content. HTML forms the backbone of web pages, allowing browsers to display and organize content in a readable format.

2.What is the difference between baking powder and baking soda?

Baking powder and baking soda are both leavening agents used in baking, but they work differently. Baking soda (sodium bicarbonate) requires an acid, like lemon juice or vinegar, to activate it and produce carbon dioxide, which helps dough rise. Baking powder contains baking soda along with an acidifying agent and a drying agent (usually cornstarch), so it only needs moisture to activate and release carbon dioxide. Baking powder can be single-acting (activated by moisture) or double-acting (activated by moisture and heat), providing more leavening power.
